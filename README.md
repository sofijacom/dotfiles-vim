> _Install_

```
git clone https://github.com/sofijacom/dotfiles-vim.git ~/dotfiles-vim
cd ~/dotfiles-vim
mv .vim ~/.vim
mv .vimrc ~/.vimrc
rm -rf ~/dotfiles-vim
```

> vim.desktop

```
[Desktop Entry]
Type=Application
Name=Vim
Comment=Edit text files
Exec=vim-normal
Terminal=true
Icon=gvim
Categories=Utility;TextEditor;
```
